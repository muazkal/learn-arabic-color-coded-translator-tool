# Arabic–English Text Aligner

An interactive tool for aligning Arabic text with natural English translations at the word level, with color-coded correspondence.

## Features

- **Arabic text input** — paste or type Arabic text for processing
- **AI-powered translation** — generates fluent, idiomatic English (not literal gloss)
- **Word-level alignment** — maps each Arabic word/phrase to its English semantic unit
- **Color-coded output** — visual correspondence between Arabic and English using color highlights
- **Editable translations** — review and refine AI-generated translations before finalizing
- **Export options** — download results as HTML or CSV

## How It Works

1. **Input** — Enter Arabic text
2. **Review & Edit** — AI produces word-level alignment with natural English; edit as needed
3. **Output** — View color-coded Arabic↔English alignment; download in your preferred format

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Lovable Cloud (backend functions & AI)
